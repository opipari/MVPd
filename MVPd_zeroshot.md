### Zero-Shot Splitting

#### CLIP Environment:

Setup environments:

```
python3.8 -m venv ./envs/clip && \
  source ./envs/clip/bin/activate && \
    pip install -r ./requirements/clip.txt && \
      deactivate
```



#### Script Command:

```
source envs/clip/bin/activate
python scripts/zero_shot_split.py
```
